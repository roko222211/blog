# Moć MDX-a

MDX je izvrstan za tehničke blogove jer omogućava **embedding React komponenti** direktno u Markdown sadržaj.

## Primjeri korištenja

### 1. Liste i formatiranje

MDX podržava standardni Markdown:

- Stavka 1
- Stavka 2
- Stavka 3

}
