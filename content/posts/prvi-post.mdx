# Dobrodošli!

Ovo je moj prvi blog post. Ovaj blog je izgrađen koristeći moderne tehnologije:

- **Next.js 15** – React framework s App Routerom
- **MDX** – Markdown s React komponentama
- **Tailwind CSS** – Utility-first CSS framework
- **Shadcn UI** – Elegantne i pristupačne komponente

## Zašto MDX?

MDX omogućava pisanje standardnog Markdown-a, ali s moćnošću React komponenti. To znači da mogu uključiti:

- Interaktivne grafikone
- Prilagođene komponente
- Dinamički sadržaj
